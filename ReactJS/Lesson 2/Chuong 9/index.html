      • ========================================================
      • react-app-rewired
      • =====> How to rewire your create-react-app project
      • 1. Install react-app-rewired (đã cài ở trên -> bỏ qua)
      • 2. Create a config-overrides.js file in the root directory
        • - root: config-overrides.js
        • File: config-overrides.js

          module.exports = function override(config, env) {
          //do stuff with the webpack config... br return config;
          }

      • 3. 'Flip' the existing calls to react-scripts in npm scripts for start, build and test
        • 1. Mở file package.json
        • 2. Thay thế "start": "react-scripts start", => "start": "react-app-rewired start",

          • /* package.json */
          "scripts": {
          - "start": "react-scripts start",
          + "start": "react-app-rewired start",
          - "build": "react-scripts build",
          + "build": "react-app-rewired build",
          - "test": "react-scripts test",
          + "test": "react-app-rewired test",
          "eject": "react-scripts eject"
          }
