      Mô hình Flexbox có mấy thành phần chính?

      - Flexbox có hai thành phần chính: flex-container và flex-item.

      Mô hình flexbox có bao nhiêu trục?

      - Mô hình flexbox có hai trục: main axis và cross axis. Trong đó main axis là trục chính, trục còn lại là cross axis và nó luôn vuông góc với main axis.
